Data Visualization & Power BI

Beginner to Advanced | 45-Day Schedule

45-Day Schedule

1
Week 1: Introduction to Data Visualization

Learn visualization principles, Power BI setup, and create basic visualizations.

2
Week 2: Advanced Visualization Techniques in Power BI

Explore advanced charts, interactivity, and custom visuals in Power BI.

3
Week 3: Data Transformation and Cleaning

Learn data transformation, cleaning, and merging queries in Power Query.

4-5
Week 4 & 5: Power BI DAX and Reports/Dashboards

Explore DAX functions, relationships, create calculations, and design reports and dashboards.

6
Week 6: Power BI Sharing, Collaboration, and Publishing

Publish and share reports, collaborate with teams, and understand Power BI security.

7
Week 7: Advanced Power BI Features

Integrate Power BI with Power Automate and Power Apps for enhanced functionality.

Course Information

Prerequisites: Basic understanding of data and visualization concepts.

Skills Covered: Power BI, Data Transformation, DAX, Reporting, Dashboard Design, Collaboration.

Course Outcomes: Master Power BI for data visualization, transformation, reporting, and collaboration.

Modules

Introduction to Data Visualization

Learn the principles of data visualization, importance, and types of visualizations.

  • Introduction to Data Visualization
  • Data Preparation for Visualization
  • Power BI Setup and Interface
Advanced Visualization Techniques

Create advanced charts and use interactivity in Power BI reports.

  • Advanced Charts
  • Conditional Formatting
  • Working with Multiple Data Sources
Data Transformation and Cleaning

Learn how to clean and transform data in Power Query Editor.

  • Removing Errors and Filtering Data
  • Handling Nulls and Duplicates
  • Creating Calculated Columns and Measures
Power BI DAX (Data Analysis Expressions)

Learn how to use DAX functions for data analysis and calculations.

  • Introduction to DAX
  • Time Intelligence Functions
  • Creating Relationships and Filters

Final Project

Project Description: Build and present a complete Power BI dashboard with interactive elements for business insights:

  • Design and organize interactive dashboards
  • Incorporate key performance indicators (KPIs)
  • Publish and share dashboards for collaboration
